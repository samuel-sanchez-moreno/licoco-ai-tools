# Scripts

Utility scripts for installing, updating, listing, and running LiCoCo AI tools.

## Available scripts

| Script | Description |
|--------|-------------|
| [`install.sh`](./install.sh) | Install a skill, agent, plugin, or workflow by name |
| [`marketplace-list.sh`](./marketplace-list.sh) | List all tools available in the marketplace |
| [`marketplace-update.sh`](./marketplace-update.sh) | Update all installed tools to their latest versions |
| [`run-workflow.sh`](./run-workflow.sh) | Run a workflow by name |

## Usage

Make scripts executable before first use:

```bash
chmod +x scripts/*.sh
```

Then see each script's `--help` output for detailed usage.
